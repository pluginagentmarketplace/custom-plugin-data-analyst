## How to Use

```
/marketplace
```

This will:
1. Show featured plugins
2. Allow searching plugins
3. Show plugin details and ratings
4. View user reviews
5. Install selected plugins
6. Manage installed plugins

## Installation

```
1. Run /marketplace
2. Search or browse plugins
3. Click "Install" on plugin card
4. Follow setup instructions
5. Start using the plugin
```

## Managing Plugins

- **View Installed**: See all installed plugins
- **Update Plugins**: Check for updates
- **Remove Plugin**: Uninstall unwanted plugins
- **Rate Plugin**: Share your feedback
- **Recommend**: Suggest to other users
