# /explore

Explore all 7 specialist agents and their expertise areas.

## Available Agents

### 1️⃣ **Foundations & Fundamentals Specialist**
Master data analysis basics, Excel, Google Sheets, and data collection.
- Excel formulas, pivot tables, data cleaning
- Google Sheets collaboration and cloud features
- Data sources, APIs, and ingestion methods
- **Best for**: Complete beginners

### 2️⃣ **SQL & Databases Expert**
SQL mastery, database design, and query optimization.
- SELECT, WHERE, JOIN, GROUP BY, and aggregations
- Window functions, CTEs, and advanced queries
- Database normalization, indexing, and optimization
- **Best for**: Data retrieval and manipulation

### 3️⃣ **Statistics & Statistical Analysis Specialist**
Statistical foundations, hypothesis testing, and probability.
- Descriptive and inferential statistics
- Probability distributions and hypothesis testing
- A/B testing, confidence intervals, and p-values
- **Best for**: Data validation and testing

### 4️⃣ **Visualization & BI Tools Architect**
Dashboard design, Power BI, Tableau, and data storytelling.
- Power BI and Tableau mastery
- Dashboard design and best practices
- Data storytelling and stakeholder communication
- **Best for**: Data presentation and business insight

### 5️⃣ **Python & R Programming Expert**
Programming for data analysis using Python and R.
- Python basics, pandas, NumPy
- R language, dplyr, ggplot2
- Data manipulation and automation
- **Best for**: Advanced data processing

### 6️⃣ **Advanced Analytics & Modeling Specialist**
Predictive modeling, regression, and machine learning basics.
- Linear and logistic regression
- Tree models and ensemble methods
- Predictive modeling and forecasting
- **Best for**: Advanced analytics and modeling

### 7️⃣ **Career Development & Communication Coach**
Career progression, communication skills, and leadership.
- Portfolio building and job search strategy
- Data storytelling and presentations
- Advancement and leadership development
- **Best for**: Career growth and communication
