# /assess

Evaluate your knowledge and get personalized recommendations.

## Overview

Use this command to:
- Assess your current skill level
- Get knowledge gaps identified
- Receive personalized recommendations
- Find next learning steps
- Track improvement over time

## Assessment Categories

### 1. Self-Assessment
Rate your proficiency in different areas:
- Fundamentals (0-10)
- Core Concepts (0-10)
- Advanced Topics (0-10)
- Project Experience (0-10)
- Best Practices (0-10)

### 2. Technical Quiz
Answer questions to evaluate:
- Language Proficiency
- Framework Knowledge
- Design Patterns
- Problem-Solving
- Architecture Understanding

### 3. Project Evaluation
- Complexity of projects completed
- Technologies used
- Problem-solving approach
- Code quality
- Impact and results

### 4. Experience Level
- Years of experience
- Roles held
- Companies/Projects
- Technologies mastered
- Leadership experience

## How Assessment Works

```
1. Answer questions about your background
2. Evaluate your proficiency in key areas
3. Assess project experience
4. Review completed learning paths
5. Generate skill report
```

## Assessment Output

You'll receive:

✅ **Current Level** - Junior/Mid/Senior/Principal
✅ **Strengths** - What you excel at
✅ **Gaps** - Areas needing improvement
✅ **Recommendations** - Suggested learning paths
✅ **Next Steps** - Specific skills to focus on
✅ **Timeline** - Estimated time to reach goals
✅ **Resources** - Relevant agents and skills

## Example Report

```
Current Level: Mid-Level Frontend Developer

Strengths:
✅ React expertise (Advanced)
✅ CSS/Styling (Advanced)
✅ Component design (Intermediate)

Gaps:
❌ TypeScript (Basic)
❌ Performance optimization (Basic)
❌ Testing (Intermediate)

Recommendations:
1. Deepen TypeScript knowledge (4 weeks)
2. Master performance optimization (3 weeks)
3. Improve testing practices (3 weeks)
4. Build advanced React projects (8 weeks)

Estimated time to Senior level: 6 months
```

## Usage

```
/assess
```

Then:
1. Answer assessment questions
2. Provide project experience details
3. Rate your skill levels
4. Receive personalized report
5. Get action plan

## Re-Assessment

Track your progress:
- Reassess every 1-3 months
- Compare against previous results
- Celebrate achievements
- Adjust learning path
- Set new goals

## Next Steps After Assessment

1. Review recommendations
2. Use `/learn` for structured path
3. Access relevant skills
4. Build projects
5. Reassess progress
