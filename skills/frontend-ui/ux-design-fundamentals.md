# UX Design Fundamentals

## UX Design Principles

### 1. User-Centered Design
- Research user needs
- Create personas
- User testing
- Feedback incorporation

### 2. Accessibility (WCAG 2.1)
```html
<!-- Semantic HTML -->
<nav aria-label="Main navigation">
  <a href="/">Home</a>
  <a href="/about">About</a>
</nav>

<!-- Color contrast (7:1 recommended) -->
<button style="color: #ffffff; background: #0066cc;">Accessible</button>

<!-- ARIA labels -->
<input aria-label="Search" type="search">
```

Create professional designs:

```
1. Create frame (mobile/desktop)
2. Design components
3. Add constraints
4. Create variations
5. Export for developers
```

### Design Handoff

- Responsive behavior

## User Research

### Methods
- User interviews (1:1)
- Surveys (quantitative)
- Usability testing (5 users minimum)
- Analytics analysis
- A/B testing

### Creating Personas
```
Name: Jane
Role: Product Manager
Goals: Increase productivity
Pain Points: Time management
```

## Accessibility Checklist

✅ Keyboard navigation
✅ Color contrast (4.5:1 minimum)
✅ ARIA labels for buttons
✅ Form labels
✅ Alt text for images
✅ Focus styles
✅ Heading hierarchy
✅ Skip links
